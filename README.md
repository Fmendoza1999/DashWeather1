# Weather Globe – World Explorer

An interactive weather dashboard that lets users spin a 3D globe, click any country, and see live weather, cultural highlights, city guides, and flight info.

**Live Demo:** [dash-weather1.vercel.app](https://dash-weather1.vercel.app)

## Features

- Interactive 3D globe (Leaflet.js)

- Flight info and news feed
